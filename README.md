## การทำงานของแต่ละส่วน

### HTML — โครงสร้างหน้าเว็บ (`index.html`)

แบ่งเนื้อหาเป็น 4 ส่วนหลัก:

1. **Nav** — แถบเมนูด้านบน มีโลโก้และปุ่มสลับโหมดมืด/สว่าง
2. **Hero** — ส่วนเปิดหน้า แสดงชื่อ ตำแหน่ง และรูปโปรไฟล์
3. **Info** — ข้อมูล 2 คอลัมน์ (เกี่ยวกับฉัน + งานที่ทำ)
4. **Card** — เหตุผลที่อยากเข้า DevPool พร้อมปุ่มอ่านเพิ่มเติม

HTML ทำหน้าที่กำหนด **"มีอะไรบ้าง"** บนหน้าเว็บ โดยใช้ semantic tag เช่น `<nav>`, `<header>`, `<section>`, `<footer>` เพื่อให้โค้ดอ่านเข้าใจง่ายและเป็นมิตรกับ SEO

### CSS — หน้าตาและ Layout (`style.css`)

หลักการสำคัญที่ใช้:

- **CSS Variables** — ประกาศสีไว้ใน `:root` ที่เดียว แล้วเรียกใช้ด้วย `var(--ชื่อ)` ทั่วทั้งไฟล์ ทำให้เปลี่ยนธีมสีได้ง่ายโดยแก้แค่จุดเดียว
- **Flexbox** — จัดของเป็นแถวแนวนอน ใช้ใน Nav (โลโก้ซ้าย ปุ่มขวา) และเหตุผลแต่ละข้อ (เลข + ข้อความ)
- **CSS Grid** — แบ่ง layout เป็นคอลัมน์ ใช้ใน Hero (ข้อความ + รูป) และ Info (2 คอลัมน์)
- **Dark Mode** — เมื่อ `<body>` มีคลาส `dark` จะเปลี่ยนค่า CSS Variables เป็นชุดสีมืด ทำให้ทั้งเว็บเปลี่ยนสีอัตโนมัติ
- **Responsive** — ใช้ `@media (max-width: 720px)` เปลี่ยน layout จาก 2 คอลัมน์เป็น 1 คอลัมน์บนจอมือถือ

### JavaScript — การโต้ตอบ (`script.js`)

มี interaction 2 จุด:

1. **ปุ่มสลับโหมดมืด/สว่าง** — ใช้ `classList.toggle('dark')` สลับคลาสบน `<body>` เมื่อคลิก ทำให้ CSS ชุด `body.dark` ทำงาน/หยุดทำงาน
2. **ปุ่มอ่านเพิ่มเติม** — ใช้ `classList.toggle('hidden')` สลับการแสดง/ซ่อนกล่องเนื้อหาเพิ่มเติม

ทั้งสองจุดใช้หลักการเดียวกัน: **ดึง element ด้วย `getElementById`** → **ฟังเหตุการณ์ด้วย `addEventListener`** → **สลับคลาสด้วย `classList.toggle`**
